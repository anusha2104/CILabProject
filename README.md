## Project Overview
This project demonstrates the implementation of Continuous Integration (CI) using Jenkins.
The CI pipeline automatically builds the project, executes unit tests, and generates artifacts
whenever code is pushed to the GitHub repository.

---

## Tools & Technologies
- Jenkins
- Git & GitHub
- Apache Maven
- Java
- JUnit
- Windows OS

---

## CI Workflow
1. Developer pushes code to GitHub
2. Jenkins automatically detects changes
3. Source code is checked out
4. Maven build is triggered
5. Unit tests are executed
6. Build artifacts are generated
7. Build status is reported

---

## Jenkins Jobs Implemented

### 1. Freestyle Job
- GitHub SCM integration
- Maven build execution
- JUnit test reporting
- Artifact archiving
- Build status monitoring

### 2. Multibranch Pipeline
- Jenkinsfile-based pipeline
- Automatic branch discovery
- CI triggered on commits
- Pipeline stages:
  - Checkout
  - Build & Test
  - Deploy
